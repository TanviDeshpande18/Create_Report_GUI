Validation parameters of NGS accuracy

An internal positive control of ɸx174 phage DNA is sequenced along with the user provided samples. Since this is a sample with a known DNA sequence, the following measurements are carried out

  1. Per base sequencing quality as assigned by the instrument
  2. Per base sequencing accuracy calculated independently
  3. Per base sequencing coverage calculated independently

ɸx174 is a synthetic internal positive control on the Illumina NextSeq 550 platform. It is provided by the manufacturer for this specific purpose. The product details can be found here: PhiX Sequence internal control V3

The product used has the catalogue # : FC-110-3001

Sequencing statistics of spike in sample

We included a ɸx174 phage DNA as a spike in control at 1% of the sample library amount in picograms.

The synthetic spike in control has no index and thus can not be demultiplexed spuriously into any data belonging to any experimental sample in the run. The ‘fastq’ reads that belong to the ɸx174 phage DNA are thus obtained as an insulated dataset and thus this analysis is independently performed without the possibility of admixture with the samples.

Sequencing quality - per base quality

The Illumina NextSeq 550 assigns a quality score to every read.

This is an estimation of the accuracy of base calling while performing the sequencing by synthesis reaction in the flow-cell. This value is measured by the instrument while reading every synthesis reaction.

The quality values are reported along with the data. The values are extracted from the data using a tool called fastqc.

Figure 1: Per base sequencing quality


This figure shows the per base sequencing quality of the reads as calculated using the as per the Illumina standard as defined here

The figure depicts average per base sequencing quality as a boxplot of values at each base in the read. This serves as a guide for pre-processing of the reads for quality filtration and read trimming.

A phred33 score over 30 implies a base call accuracy of 99.9%. For all the positions, 90% of the reads have a quality score above Q30 and thus pass the QC check for further analysis
